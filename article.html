Working with Vim: Essential Commands

Vim is a powerful text editor used primarily in command-line interfaces. It allows efficient text editing and offers a range of features for coding and writing. Below are some key commands that I have used recently to work on my projects:

Basic Commands

  • vim file.html - Open or create a file named file.html.
  • i - Enter insert mode to start editing the file.
  • Esc - Exit insert mode.
  • :wq - Save changes and exit the file.
  • git add . - Stage all changes for the next commit in Git.
  • git commit -m "message" - Commit staged changes with a message.
  • git push origin main - Push committed changes to GitHub.

Vim may seem complex at first, but with practice, it becomes a powerful tool for productivity, especially when working with Git and editing files from the terminal.
